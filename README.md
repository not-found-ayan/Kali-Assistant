# Kali-Assistant
Get help about execution of any application command, and quick in app execution!

## How to Install:
### Step 1: Save the Core Application Code
Download the kali_assistant.py and install.sh files (put them in same directory).

### Step 2: Build the Installer Script
To package this application and place it directly into the Kali Linux Desktop menu (so you can launch it like any other program).

### Step 3: Run the Installer
Make the files executable and run the installation script:

cd <ur-file-path>
chmod +x kali_assistant.py install.sh
sudo ./install.sh

## How to Run:
Because the installer registered the program system-wide, you have two convenient ways to open it:

    From the Desktop Menu (GUI):

        Open your Kali Application Menu (top-left dragon icon).

        Search for "Forest Terminal Assistant".

        Click the icon to launch it directly.

    From any Terminal:
    Open a terminal window and simply type:
    code Bash

    kali-assistant

    (To run it in the background so it doesn't block your current terminal, use kali-assistant &).
